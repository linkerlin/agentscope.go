Chat Demo - Agents with Auto Tools

Because this Studio server runs with AutoStandardTools + auto TaskStore + Workspace, the agent you chat with has automatic file/workspace tools, task management, web/json utilities etc. Try asking it to "list the workspace files" or "create a task about testing auto assembly".

This demo uses the v2 chat endpoint. Because the server was started with AutoStandardTools + Workspace, any agent you chat with will have automatic access to file tools, tasks (auto in-mem store), glob/grep, web_fetch, json tools, etc. in their sessions.

Real SSE streaming from /v2/chat/stream. Tool calls from auto-assembled tools (file, task, web, json) will be shown in UI. First register demo user on Dashboard if auth required.
