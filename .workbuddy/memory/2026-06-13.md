## GoGPU vs Born 深度对比研究
- 应步子哥要求，对 `gogpu/gogpu` 和 `born-ml/born` 两个 GitHub 仓库进行全维度深度对比
- 产出完整对比报告：`GoGPU_vs_Born_深度对比报告.md`（14 章节，含架构图、功能矩阵、性能基准、依赖关系）
- **核心发现**：
  - GoGPU (v0.41.9, 305☆, 163 releases) 是 GPU 图形/计算底层基础设施，Born (v0.9.1, 96☆, 35 releases) 是其下游深度学习框架
  - 二者非竞品而是上下游互补：Born 依赖 gogpu/wgpu 和 gogpu/naga
  - GoGPU 覆盖全平台全后端（Vulkan/DX12/Metal/GLES/Software），Born 仅支持 WebGPU/D3D12
  - Born 的 LLM 功能亮眼：Flash Attention 2、KV-Cache (3.94x)、推测解码 (2-4x)、GGUF/ONNX 导入
  - Born 最大风险：社区极小、GPU 后端单一、CPU 无 SIMD 优化
