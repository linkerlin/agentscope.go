## 三、功能实现详情

### 3.1 PersonalSummarizer (个人记忆提取器)

**核心功能**:
- ✅ 从对话中提取观察记忆 (时间相关过滤)
- ✅ 从观察中总结深层洞察
- ✅ 基于新观察更新已有洞察
- ✅ 处理矛盾/重复记忆 (LLM判断)

**算法亮点**:
```go
// Jaccard相似度计算洞察与观察的相关性
func (s *PersonalSummarizer) calculateJaccardSimilarity(set1, set2 map[string]struct{}) float64

// 关键词提取和过滤
func (s *PersonalSummarizer) extractKeywords(text string) map[string]struct{}
```

**与Python对齐**:
- GetObservationOp → ExtractObservations
- GetReflectionSubjectOp + UpdateInsightOp → ExtractInsights + UpdateInsights
- ContraRepeatOp → HandleContraRepeat

### 3.2 ProceduralSummarizer (程序记忆提取器)

**核心功能**:
- ✅ 从执行轨迹提取任务经验
- ✅ 成功模式提取 (Score ≥ 0.9)
- ✅ 失败教训提取 (Score < 0.5)
- ✅ 改进建议提取 (中间评分)
- ✅ 多条成功轨迹对比学习
- ✅ 记忆质量验证

**轨迹评分策略**:
```go
if traj.Score >= 0.9 {
    // 提取成功模式
} else if traj.Score < 0.5 {
    // 提取失败教训
} else {
    // 提取改进建议
}
```

**数据结构**:
```go
type Trajectory struct {
    Messages []*message.Msg
    Score    float64  // 成功率
    TaskName string
}
```

**核心功能**:

- ✅ 使用指南生成 (基于最近30条记录)
- ✅ 最佳实践文档生成

5. 统计信息 (成功率、平均耗时等)

### 3.4 MemoryDeduplicator (记忆去重器)

**核心功能**:
- ✅ 向量相似度去重 (余弦相似度)
- ✅ LLM语义去重 (可选)
- ✅ 与存储去重 (防止重复添加)
- ✅ 矛盾检测
- ✅ 相似记忆合并

**去重流程**:
```
记忆列表
    ↓
[向量相似度去重] ← threshold: 0.85
    ↓
[LLM语义去重] (可选)
    ↓
[矛盾检测]
    ↓
[相似记忆合并]
    ↓
去重后的记忆列表
```

**相似度阈值**:
| 阈值范围 | 处理策略 |
|---------|---------|
| 0.95+ | 视为完全重复，删除 |
| 0.85-0.95 | 高度相似，可合并 |
| 0.70-0.85 | 相似，保留但标记相关 |
| < 0.70 | 不同，独立保留 |

## 五、使用示例

### 5.1 个人记忆工作流

```go
func PersonalMemoryWorkflow(ctx context.Context, vectorMem *ReMeVectorMemory, model model.ChatModel, msgs []*message.Msg, userName string) error {
    // 1. 创建提取器
    summarizer := NewPersonalSummarizer(model, "zh")
    
    // 2. 提取观察
    observations, err := summarizer.ExtractObservations(ctx, msgs, userName)
    if err != nil {
        return err
    }
    
    // 3. 处理矛盾/重复
    filtered, _, err := summarizer.HandleContraRepeat(ctx, observations)
    if err != nil {
        return err
    }
    
    // 4. 提取洞察
    insights, err := summarizer.ExtractInsights(ctx, filtered, userName)
    if err != nil {
        return err
    }
    
    // 5. 合并并去重
    allMemories := append(filtered, insights...)
    deduplicator := NewMemoryDeduplicator(vectorMem.store.embed)
    unique, _, err := deduplicator.Deduplicate(ctx, allMemories)
    if err != nil {
        return err
    }
    
    // 6. 存储到向量库
    for _, mem := range unique {
        if err := vectorMem.AddMemory(ctx, mem); err != nil {
            return err
        }
    }
    
    return nil
}
```

## 八、已知限制

1. **嵌入模型依赖**: 当前实现依赖外部嵌入模型，需要有效的 API Key
2. **LLM 调用成本**: 记忆提取需要多次 LLM 调用，有一定成本
3. **中文支持**: 主要优化了中文场景，其他语言需要验证
4. **并发性能**: 大规模并发场景需要进一步测试优化
